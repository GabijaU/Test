HTML Table Reference Cheat Sheet

Table Tags

Tag Name Description
<table> Table An element having content that is used to represent a two-dimensional table made of rows and columns.
<thead> Table Head Defines the headings of table columns encapsulated in table rows.
<tr> Table Row Used to add rows to a table before adding table data and table headings.
<th> Table Heading Used to add titles to rows and columns of a table and must be enclosed in a table row element, <tr>.
<td> Table Data The table data element can be nested inside a table row element, <tr>, to add a cell of data to a table.
<tbody> Table Body The set of rows containing actual table data other than table heading and footer.
<tfoot> Table Foot Uses table rows to give footer content or to summarize content at the end of a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Random Info

To show HTML tags as text, use & lt; (without space) to represent <, and & gt; (without space) to represent >.
